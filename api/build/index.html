    Properties

    name

    模型名称

    字符串

    TAG

    模拟信息记录，用于记录和输出信息

    TAG是一个结构体，可利用其存储各类输出数据

    SET

    模拟设置信息，可记录二次开发新的数据

    SET是一个结构体，可利用其存储各类二次开发数据

    PAR

    用于平行计算的参数

    平行计算未启用

    GROUP

    组信息，详见下方说明

    记录了定义的各个组，组名不能以group开头

    Loads

    荷载信息

    未启用，未来可能去除

    isPar

    是否并行计算

    功能未启用

    is2D

    是否为二维模型

    默认为0，进行三维计算

    data

    结果数据集

    部分后处理命令的数据集，如位移场、应力场（见右说明）

    mo

    计算模块

    po

    孔隙对象，用于孔隙渗流和流固耦合计算

    见PoreFlood等示例

    status

    模型状态

    Mats

    材料元胞数组（cell array）

    材料的元胞数组，d.Mats{1}为第一个材料

    vRate

    粘滞阻尼系数

    临界阻尼所要乘以的系数，见文字教程中阻尼的说明

    g

    重力系数

    默认为-9.8

    module

    此模拟的建模器，如obj_Box

    未启用，未来可能去除

    isUI

    是否在窗口程序中运行，取0时，show命令将在新窗口画图

    默认为1

    edit_output

    提示信息对象

    系统参数，勿修改

    aNum*

    模型单元和固定墙单元个数和（见下方具体说明）

    mNum*

    模型单元个数（见下方具体说明）

    aMatId

    单元的材料编号

    aX

    模型单元的X轴坐标

    aY

    模型单元的Y轴坐标

    aZ

    模型单元的Z轴坐标

    aR

    模型单元的半径

    aKN

    单元正向劲度系数

    aKS

    单元切向劲度系数

    aBF

    单元断裂力

    aFS0

    单元初始抗剪强度

    aMUp

    单元摩擦系数

    isFluidVis

    作用在单元上是否为流体的阻尼设定，运行groupMat2Model或applyMaterial后，这一参数产生效果，这一设定增加于2.60版

    以落体为例，当设为流体阻尼时（1），单元的阻尼力与单元面积成正比，那么大的颗粒的下沉速度较快；当设为固体阻尼时（0），阻尼力与单元体积成正比，大颗粒和小颗粒的下沉速度一致，类似于用一个大颗粒代表相应的小颗粒时，总阻尼也是与体积成正比

    mVis

    单元粘滞阻尼系数

    mM

    单元质量

    period

    一个单元固定在墙上时的简谐振动周期

    dbXYZ

    单元在newStep函数中的边界位移

    bIndex

    边界单元信息，功能被取代，未来将取消

    saveHour

    保存数据文件间隔时间

    step

    当前模拟步，用于迭代计算，见示例

    totalStep

    总模拟步

    isNote

    是否显示模拟提示

    0，1

    note

    提示内容

    figureNumber

    绘图的窗口号，可指定在某一窗口绘图

    见Roller3，PoreFlood3等示例

    Rrate

    显示单元的半径系数

    取0.5可看在内部颗粒，默认为1

    showB

    显示边界方式

    可为0，1，2，3，具体见后处理

    showBallLimit

    大于此值时，单元显示为点

    默认为一千万

    Methods

    addTimeProp(gName, type, Ts, Vs)

    设置指定组的属性随时间的变化，可用于产生振动。设置信息记录于d.mo.SET.PROP

    gName:组名；type:属性名；Ts:时间；Vs：时间对应的值

    无

    见user_LineModel

    removeTimeProp(gName, type)

    删除指定组的属性随时间的变化

    gName:组名；type:属性名；

    无

    见user_LineModel

    addRecordProp(gName, type)

    设置要在运行d.recordStatus时，记录指定组的属性平均值，设置信息记录于d.status.SET.PROP。仅限于记录d.mo中的单元属性，其它属性可使用d.status.recordCommand来设置和使用，具体见LineModel示例

    gName:组名；type:属性名；

    无

    见user_LineModel

    removeRecordProp(gName, type)

    删除指定组的属性记录

    gName:组名；type:属性名；

    无

    见user_LineModel

    addElement(matId, addObj, varargin)

    利用结构体（.X,.Y,.Z,.R信息）增加单元，增加单元后会自动运行d.setStandarddT将时间步设为标准值

    材料号，结构体或多个结构体的cell数组，第三个参数可选，为单元类型，取"model"（活动单元）,"wall"（固定单元)，默认为"model"

    [sphere1Id,sphere2Id,boxId]=d.addElement(1,{sphereObj,sphereObj2,boxObj});

    addFixId(direction, gId)

    增加要锁定自由度的单元（锁定坐标）

    锁定的方向（'X','Y','Z'，其三个方向的组合)；gId:单元编号数组

    此维度上新锁定编号newFixId

    d.addFixId('X',2);锁定第二个单元的x坐标，d.addFixId('XYZ',2);锁定第二个单元的xyz坐标

    addGroup(gName,gId,varargin)

    在当前模型中定义一个新组，d.addGroup(gName,gId,matId);

    gName:组名；gId:单元编号数组; matId：材料号。可为2或3个输入参数，当输入参数为2个时，材料号默认为1，可采用d.setGroupMat来修改组的材料

    无

    d.addGroup(gName,gId);

    newMat:材料对象

    无

    最重要的平衡迭代函数，见下方说明

    多个输入参数

    balanceBondedModel(varargin)

    赋材料后，自动胶结和平衡模型，有单元间摩擦力。原始松散无粘结的堆积体将被粘结

    输入参数为标准平衡次数，为空时则取1

    无

    参考BoxModel等示例的第二步

    balanceBondedModel0(varargin)

    赋材料后，自动胶结和平衡模型，无单元间摩擦力。原始松散无粘结的堆积体将被粘结

    输入参数为标准平衡次数，为空时则取1

    无

    参考BoxModel等示例的第二步

    balanceForce(Amax, num)

    平衡模型中的力

    Amax:目标最大加速度，num:平衡步次

    无

    breakGroup(varargin)

    断开指定组内连接或两个组间的连接

    无输入参数时，d.breakGroup()，断开所有单元的连接；一个输入参数时，d.breakGroup('group1')则断开这个组的组内连接，如输入的是元胞矩阵，d.breakGroup({'group1','group2'})，则断开这些组的组内连接；如输入二个参数时，d.breakGroup('group1','group2')断开这两个组之间的连接

    nBreakFilter

    d.breakGroup('sample');

    breakGroupOuter(varargin)

    断开指定组向外的连接
